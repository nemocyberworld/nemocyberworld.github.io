# ⚡ NemoCyberWorld - Default Landing Page

Welcome to the official landing page of **NemoCyberWorld** — a sleek, dark-themed gateway to your cybersecurity learning journey.

This project serves as the **front door** to the NemoCyberWorld ecosystem. Built using HTML and Tailwind CSS, it’s designed for clarity, accessibility, and hacker-style aesthetics that set the tone for what’s inside.

---

## 🌐 Overview

The **NemoCyberWorld Landing Page** is a static HTML interface that:

- Welcomes users to the platform
- Introduces the purpose and mission
- Provides a sidebar for navigation (extensible)
- Offers a clean, responsive, and modern look using Tailwind CSS
- Prepares for expansion into an interactive learning dashboard

To use or customize the landing page:

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Customize the layout, sidebar, or main content as needed.

```bash

cd nemocyberworld-landing
open index.html
```
